# Alternative Liste Gemeinderatswahl - Template

Template-Seite für Kandidat:innen der Alternative Liste zur Gemeinderatswahl.

## Features

- Responsive Design
- SEO-optimiert mit strukturierten Daten
- React Router für URL-basierte Navigation
- Wahlprogramm-Übersicht mit Illustrationen
- Social Media Integration

## Setup

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm start
```

## Konfiguration

Kandidatendaten in `src/data/candidate.json` anpassen:
- Name, Wahlkreis
- Über mich Text
- Social Media Links

Wahlprogramm in `src/data/wahlprogramm.json` bearbeiten.

## Tech Stack

- React 18
- React Router
- Styled Components
- Vite
- Express (Production Server)
